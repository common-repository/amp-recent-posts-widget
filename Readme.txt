Generates a list of AMP posts in the widget area.


== Description ==

__NOTICE: There are currently better options available, please check our plugins [AMP Recent Posts](https://wordpress.org/plugins/amp-recent-posts/) and [AMP Footer Widgets](https://wordpress.org/plugins/amp-footer-widgets/), this plugin is now obsolete and will eventually be removed from the repository.__



== Installation ==

1. Upload and activate the plugin.
2. Go to 'Widgets' and place the widget in a sidebar.

That's it!


Customizing the output of the widget:

1. __Slug__: depending on which AMP plugin you are using you can adjust the slug to direct to the AMP articles.
Standard: 'amp' (appropriate for the AMP plugin by Automatic). 

Example: 'amp-post' will generate a slug 'yourwebsite.nl/post/__amp-post__/'

2. __Title__: add a H4 widget heading.
Standard: none
3. __Total links__: set a limit to the amount of posts displayed.
Standard: 5000
4. __Date__: add the post date.
Standard: none
